# Ulangan-Praktik-Live-Coding
Aplikasi untuk merekam seberapa konsisten siswa dalam belajar dan memberikan rekomendasi jadwal belajar serta saran teknik belajar.

## Fitur
- Input waktu belajar harian dengan mapel
- Menghitung konsistensi belajar (total hari, total jam, rata-rata per hari, hari aktif mingguan)
- Rekomendasi jadwal berdasarkan pola belajar
- Saran teknik belajar umum
- Lihat daftar mapel yang telah dipelajari

## Cara Menjalankan
1. Pastikan Python 3 terinstall.
2. Jalankan `python main.py`
3. Ikuti menu untuk input data atau lihat laporan.

## Struktur Data
Data disimpan dalam `study_data.json` sebagai list of dict dengan 'date' dan 'duration'.
